# Portfolio Bitrix24

Портфолио проектов по внедрению, развитию и сопровождению **Bitrix24**.

## Основной фокус

- CRM и автоматизация бизнес-процессов.
- Смарт-процессы и цифровые рабочие места.
- Интеграции через REST API, вебхуки и batch.
- Нестандартные сценарии автоматизации.
- BI-аналитика и отчётность (SQL, Trino).
