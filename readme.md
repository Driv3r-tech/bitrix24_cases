# Portfolio Bitrix24

Портфолио проектов по внедрению, развитию и сопровождению **Bitrix24**.

## Основной фокус

- CRM и автоматизация бизнес-процессов
- смарт-процессы и цифровые рабочие места
- интеграции через REST API, вебхуки и batch
- нестандартные сценарии автоматизации
- BI-аналитика и отчётность (SQL, Trino)

## Избранные кейсы

| Кейс | Что реализовано | Результат |
|------|------------------|-----------|
| [Автоматизация платежей в юридической компании](./portfolio/bezzabot.md) | Графики платежей, генерация документов, интеграции, контроль финансовых обязательств | Прозрачность финансов и снижение ручных ошибок |
| [Автоматическое распределение лидов в туристической компании](./portfolio/cb.md) | Очереди распределения, лимиты, контроль дублей, ночная логика перераспределения | Быстрое назначение лидов и равномерная нагрузка |
| [Миграция 4 порталов Bitrix24 в единый портал](./portfolio/fpb.md) | Аудит различий, приложение для миграции, перенос сущностей и логики CRM | Объединение порталов без потери данных |
| [CRM для производственной компании](./portfolio/ras.md) | Продажи, производство, контроль оплат, дедупликация, смарт-процессы | Сквозная цифровизация от заявки до производства |
| [Сквозная аналитика и CRM-отчётность](./portfolio/union_polymer.md) | UTM, CRM-формы, коллтрекинг, аналитика продаж и рекламы | Полная видимость пути клиента от рекламы до оплаты |
| [Автоматизация колл-центра медицинского центра](./portfolio/strategy.md) | Интеграции каналов, устранение дублей, доработка бизнес-процессов | Стабильная CRM и чистая аналитика по обращениям |
